Java Archive (JAR)

A jar file is like a zip file. In other words, it is like a suitcase/folder that contains all the source codes and contents needed to run the application. A JAR file gets created during the final phase of packaging your application.

Used In

  • Java applications are turned into Jar files for final devlivery
  • Like the Pattern Editor, it has a jar file that can be run using command prompt java -jar patternEditorFileName.jar
