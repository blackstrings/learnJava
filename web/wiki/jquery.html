Jquery is base on selectors $()

The selector is base on css and is use to tell jquery what element to target

  • $("p.test") = for classes
  • $("#test") = for id
  • $("*") = for all elements
  • $(this) = for this element
  • $("[href]") = all element with the href/li>
  • $("ul li:first") = first li of ul
