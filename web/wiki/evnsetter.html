EnvSetter (a.k.a. Environment Setter)

An internal tool that has a .bat file allowing to run it on local machine - created by our developers to easily change the environment our java apps points too. Refer to Environment pointing to get a better idea of how useful this tool is.

How to use

  • Grab the file off the F:drive
  • place the extracted zip folder anywhere
  • Run the Bat file by double clicking
