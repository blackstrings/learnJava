More Smooth Render

Subdivision

Ctrl+1 - Ctrl+5 will smooth your object. Ctrl+0 to return to no sub. This will increase smoothness in render.

Transparent Texture

you have to enable Z-transparency and set alpha of the material to '0'

then tell blender texture should affect alpha channel > turn on texture alpha and leave it at one.
the .mtl will now include d set to 0.0, and map_d set to it's alpha image.

Auto Subdivision

If your sharp edges are being rendered round, and seems like a render issue, it is probably because you have your render settings set to audo subdivide. This can happen if you accidently press the hotkey for subdividing your models

Set it to zero to go back to default, or use this as a way to subdivide all renders

UV Editor []

Where you wrap texture around 3d objects

Shft+F10 to jump to UV Editor view. use Shft+F5 to go back to standard view.

NOtE: To see any uvs, you must also load an image, Alt + O

Select your obj > edit mode > vertex > A for all > then U for unwrap, or UV > unwarp

If the object doesn't alredy have UV created, press U > choose cube projection or smart projection, then unwrap

If object does not have UV, you will not see a UV tab

Cutting UV

AKA marking Seams. select vertices like a line of vertices (Shft+Alt+LMB on a vertex) > Ctrl + E (bring up mesh menu) > Edges > and mark seams. This will to tell UV editor to cut the seam where selected.

Unwrapping UV

(optional) Ensure you have mark seams to edge using Cutting UV. Have another window with UV Editor open. Select object > Edit mode > A to select all vertices > U (unwrap menu) > unwrap. The UV Editor should show unwrapped UVs.

In UV Editor, to move uv islands, select a vertex, Press L to auto slect the island. Use (S)cale, (R)otate, (G)rab, (B)order, (A)select/deselect all to move the islands.

Stitching UVs

Project UVs first on your selected faces in edit mode > uv editor > select all visible uvs press A > Ctrl+P (pack island) > select 2 edges > stich (V key) > enter to confirm > continue uv editing

press enter to confirm after sticthing, select one edge > Press L to select island > move,rotate,scale as need

Node Editor [] node []

Can be used for material editing and as an image compositing software

To use you should have two windows up, node editor and UVEditor, or load the compositing layout.

Basic

To use nodes, you first need to enable nodes by checking use nodes in the Node Editor window. Make sure to check it off when you want to do regular 3d render from the camera views. Leaving it on and F12 will render the nodes instead.

For image, nodes work together with the UV Editor. To see the image from nodes in the UV Editor, you need to create a few things: Image, Viewer - use Shft+A > input > image and again but output > Viewer. select the node, and in UV Editor, select Viewer Node to see the image.

Adding Nodes

Make sure use nodes is check > Shft+A > search for your nodes or select from popup menu

Mix, Translate, Scale, Mask, Viewer, Image, Z Combine etc to name a few

Nodes that go together: Mask,Mix

Connecting Nodes

Blender makes it really simple to know what nodes can connect to each other. Left connectors are input and right side are outputs

Combine Images

Only one image needs to connect its Z output to the z combine, then select the viewer node to be display in UV editor. UV Editor should have viewer node selected to see it.

Composite multiple views

Method1: The trick is to duplicate the obj and turn it on its side, and render it in one layer. The render layers only work with one camera, not multiple cameras

Method2: Use two scenes, one object in each scene, then render. and you should be good. Below is more advance if you care.
Link obj1 to be in layer 1 and obj2 in layer 2 > layers tab > layer1 set to render > create a 2nd scene.

select link objects > select cam > U > Object and Data (make camera independent from other scene > adjust cam to your liking > still in new scene > layers tab > select 2nd layer to be render.

Render Layers [] Scene Layers []

Allow us to composite renders sepearetly like layers

Add Obj To Layer

Select an obj > Press M > select layer. Obj can be on multipe layers, use Shft.

Press 1,2,3, to change scene layers on the first row, Alt+1, Alt+2, ... for the 2nd row

Under Layer property context > Then select each render layer, and select the layers it can render. Pressing F12 or render, will render however many render layers that were checked. Name each render layer accordingly for less confusion.

More RenderLayers

Scene Layers

Scene layers on the other hand are just for displaying what gets display on screen for easier to work with complex scene.

Only objects in scene layer will be rendered

Objects not visible in scene layer, but selected in render layers, will not render.

Objects in scene layer but not in render layer, will still cast shadow for other objects. Use this for like shadow trick renders

Slots

Select a different slot before rendering to store the render image in a buffer
