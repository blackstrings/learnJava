Blender

A free 3d Software we can use to create 3d objects.

Object Transform Property Window(s)

Select obj and press N to bring up the transform property window.
Also the T key brings up the transform tools window

3D Cursor / crosshair left mouse

The 3d cursor that helps you change the pivot point of an object. works in all mode.

Simple test is left mouse click somewhere in 3d > shft + A > add a obj and it will be placed where 3d cursor is.

Pres N to bring up the transform property window > scroll down to find 3d cursor window

Snapping 3dCursor

Shift + C = to center grid
Shift + S > Cursor to grid

Locking with View

Problem with zooming focus camera rate with mouse wheel? Use view property and check the lock to cursor3d to control your zoom rate and view. Press N to bring up the transform panel first then look for the view window.

Snapping []

In object mode, use shft+tab to turn on snapping and again to turn off.

move object by right click and hold, then use x,y,z to further combine and snap to an axis

Create Mesh []

shortcut create

LMB to place 3d cursor in scene > Shft + A > select mesh

Upon creation, you get a little window to define its final properties before the mesh is final

Slice

edit mode > select edge > T to bring up left panel > loop cut and slide

Transformation [] Reset []

Move, rotate, scale

In blender the keys are G(grab), R(rotate), S(scale) - then combine with xyz for axis control, and shft for smaller control

Shortcut to move in axis is press G, then MMB and drag in a direction

Freeze Transformation

You are done scaling, translating, rotating, now you want object to set object to have zero values for export

Zero out object's transformation, Obj shape size, stays the same.

In blender, it is known as apply. Object mode > select object > Ctrl + A > choose one or all (one at a time) to reset

Undo Transformation

In blender, it is known as clear. Move, rotate, scale back to orginal values = Alt + G

Normals []

Recalculate All Sides

Select obj > edit mode > Face > A for all face selected

Ctrl + N = recalculate all normals face outside

Ctrl + Shft + N = recalculate all normals face inside

Flip 1 Face

Edit mode > select face > press T > shading/UVs

Object Info []

Vertex Count, edge count

Top right hand corner will display the total vertex in scene

To view individual object count, select obj > edit mode > and top right hand vertex will update to object only

Scaling 3D Obj

Basic

Object mode > S

use x,y,z to further combine and snap to an axis scale mode. Combine with numpad view 7,1,3 for better view

Reset Scale

After modifying object to scale, Ctrl+A > Scale to reset the size to 1

Camera []

Lens modification

Default is 35mm, to change focal length select the camera icon

Ortho

Select Cam > Camera Len Icon/Context > Select Ortho

Set Active Cam

When you have multiple camera, to set new camera as your numpad 0, select cam > Ctrl + 0, then use numpad 0 to toggle in view

Cam Point To Current View

Navigate normally in 3d perspective to where you want the cam to face and see > select camera in outline > Ctrl + Alt + numPad 0

or 3D View menu: View -> Align View -> Align Active Camera to View

Cam Point To Object

Add a constaint to an object, more advance, but not efficient

Render []

Align camera to view (ctrl + Alt + 0) then Press F12 to preview render, esc to close. You must manually save the file doing it this way

Render 1 frame vs animation. Clicking the action icon will render animation. Warning:It will start rendering 250 frames by default. Esc to stop and again to close render view. Change the start and end to 1 to render just one frame.

icon above is usually located at the bottom of each view

Transparent Background

Make sure you obj has a material. > output set to RGBA > Shading to Transparent

or

More Smooth Render

Subdivision

Ctrl+1 - Ctrl+5 will smooth your object. Ctrl+0 to return to no sub. This will increase smoothness in render.

Transparent Texture

you have to enable Z-transparency and set alpha of the material to '0'

then tell blender texture should affect alpha channel

UV Editor []

Where you wrap texture around 3d objects

Shft+F10 to jump to UV Editor view. use Shft+F5 to go back to standard view.

NOtE: To see any uvs, you must also load an image, Alt + O

Select your obj > edit mode > vertex > A for all > then U for unwrap, or UV > unwarp

If the object doesn't alredy have UV created, press U > choose cube projection or smart projection, then unwrap

If object does not have UV, you will not see a UV tab

Cutting UV

AKA marking Seams. select vertices like a line of vertices (Shft+Alt+LMB on a vertex) > Ctrl + E (bring up mesh menu) > Edges > and mark seams. This will to tell UV editor to cut the seam where selected.

Unwrapping UV

Ensure you have mark seams to edge using Cutting UV. Have another window with UV Editor open. Select object > Edit mode > A to select all vertices > U (unwrap menu) > unwrap. The UV Editor should show unwrapped UVs.

In UV Editor, to move uv islands, select a vertex, Press L to auto slect the island. Use (S)cale, (R)otate, (G)rab, (B)order, (A)select/deselect all to move the islands.

Display Object with Texture

Set shading to texture in view port

Base on your lighting, you'll see parts of your model shaded. Add more lights to rough out your shading. To make your texture renderable, you need to add new material to the object.

Context [] Property Panel

When you select an 3d object in view, this panel will dynamically change.

Lighting property can be modify using this panel

Material [] & Texture [] (material/texture)

Add Material

To see renderable texture, object must have material and texture setup. Select object > go to Material Context > create new material Also create a texture. You can then preview the shader in view, without having to render. After setup, you can render and will see material with texture.

Add Texture File

To use an image as texture > texture context > set type to Image or Movie > then load your file

Preview of your shader

After you add material and texture you can use the preivew window to see it

Selection []

Single/Multi Selection

Right click to select the obj, vertext, edge, face. B + LMB drag to select multiple selecitons. Shft to selected addon selections.

View [] Mode and Panel

Controls what object is lock, selectable, renderable, and can be used for selection

Toggle Camera View Mode

Use the num key pads
0 = cam/user perspective
7 = top ortho
1 = side ortho
3 = front ortho
5 = perspecitve vs ortho

. = focus cam back to center of scene if nothing is selected . = focus cam to selected object Ctrl + Shft + MMB + hold = move mouse forward and back to slowly zoom in

If zooming becomes too slow, select object in object mode, press . to reset the zoom factor

Set the active camera as main = Ctrl + 0 then use numpad 0 to change view

Toggle wireframe/x-ray

press z or see image
