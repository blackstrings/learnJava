Three.js

A 3D framework all in javascript initially proposed (2014) to be our next 3d engine framework, but Unity Engine (2015) will be become the 3d engine instead.

Features

We currently only use threejs in the shelving application, however, It was used to create prototypes until Unity came along :). FYI - ThreeJS was not built to be used in a AMD modules structure. so you can't use threejs in requirJS, you'll have to call threeJS explicitly.

ThreeJS Local Sandbox

Getting Started

download threejs.min.js from online source, there's only one file. If you download the whole package, you'll get examples, plus the loaders and good stuff. Then explicitly load the threejs script into your html. Visit the examples to see.

Three js samples

Note that Three JS does not allow loading external files on Chrome or IE, due to cross domain policy. Use firefox instead. If you procedurally create objects, then it'll work, but if you load obj files and texture, it wont load in Chrom or IE.
