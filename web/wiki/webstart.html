Webstart Apps

These are standalone apps that can be downloaded from MCOM as JNLP file and run locally on your machine. Some of the apps are deck, landscaping, and flooring.

Testing Webstart

Merica will give you links to test webstart MCRs. You test it at your local PC as it can only work here. Always have the java console open and handy when running these.


After you enable the java console, when you run a webstart app, the console will auto open for you. If you run into any issues, look at the java console logs

    Before running the JNLP download link, do the following if you are testing against stage (note this is different from preview or dub dub)
  • browser > stage.menards.com
  • Login using your user and pass
  • Point to a store with your login account and make it your store
  • You can now run the JNLP and it should pick up your store on stage
