Networking Intro

Networking Basics

Allowing multiple computers to communicate back and forth under the same network. One way computers communicate between each other is through a network, in which assigns a computer a unique IP address. It is how PC1 can locate and communicate with PC2.

Terms

IP

  • IP = Internet Protocol = example: 192.168.1.1
  • 192 is an address
  • 168 is a sub address under 192
  • and so on are sub addresses
  • You can find your ip address in command window, ipconfig

Ports

Using ports is just a way to organize the different types of communication between computers.

  • the common port for web is 8080
  • Some ports are restricted for only certain usage
  • More about ports
