Require JS

A AMD (asynchronous module definition) framework that handles dependency loading in javascript. When using requireJS, functions are no longer global, but local to each module. Use define for defining modules and pages that needs to LOAD modules, use the require.

Define

Use define when you are defining a module to be later required. Modules do not run unless a require calls them as a dependency.

Require

Use require when you are requesting or requiring that certain modules need to exist in order to interact between modules.
