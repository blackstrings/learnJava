Landscaping Visualizer

A very complicated web application based on Java and JavaServerPages (JSP). Used to estimate materials and visually see blocks change

Unique features the app supports

  • Change blocks and see it implemented automatically on screen

Setup

  • Eclipse
  • Java
  • JavaServerPages (JSP)
  • SQL
  • Database Connection

Database

Visualizer uses two database to grab product information.

  • Visualizer Database
  • DIC Database (design-it database)
  • MCOM (soon it will just use MCOM but not yet implemented)
