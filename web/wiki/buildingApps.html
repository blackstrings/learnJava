Building Apps

You can build the apps and run it outside of eclipse.

Directions

    You can use the command line or build it inside eclipse
  • Choose the command you want to build
    • mvn clean package -PstoreDeploy -Dmaven.test.skip=true
  • Login using your user and pass
  • Point to a store with your login account and make it your store
  • You can now run the JNLP and it should pick up your store on stage
