TotoiseSVN

When you get your PC, the application should already come installed with TortoiseSVN. If not talk to your mentor or teamlead.

SVN is a tool which allows multiple users to manage version-ing files by checking out and checking in files from the repository (a storage server). In other words, the same file with changes made by one person can be downloaded and viewed by another person to then be further edited and put back into the repository.

When you open ToitiseSVN, you will notice a lot of folders. You will also soon realize that multiple Menards departments share the same repository where they also put their own folders here that's not related to your team

Setup

  • To use and start checking out files, Right click inside an opened folder window > SVN Checkout > URL of repository
  • You can use either one below to start your first test
  • svn://omdc-source/ISS/Cross-Team/
  • or
  • svn://omdc-source/ISS/Kiosk/

Two Major Folders in the Repository We Care About

  • CrossTeam - utils tools shared by other departments
  • Kiosk - everything relating to our applications, even if the kiosk isn't on our team.

Terminology

  • URL of repository - The location of the repository that you want to check out.

  • Checkout directory - where you want the files from the repository to go on your local machine.

  • Checkout Depth - How much of the structure you want to check out. You can check out a single item but we always just use Fully Recursive.

  • Revision - Each change a person commits gives the structure a new revision number. Head revision means the latest and greatest, but you can select the Revision button to choose what change specifically you want to check out.

  • Clicking OK will give you the folder and files and place them where the checkout directory is specified.

  • Checkout - checkout a file

  • Commit - put the file back on the repository

Software Features

  • Revert
    • Allow a modify file to change back to original
    • right click on file/folder > tortoiseSVN > Revert
  • Checkout
    • Allow file(s)/folder(s) to be checkout and downloaded to your computer
    • right click on a folder you want > checkout SVN > provide URL of directory and destination > okay
