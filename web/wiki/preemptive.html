PreEmptive

A online analytic site we use to monitor our standalone apps. It allows us to see things like, how many users used the app, what days, how long they stayed, how many printouts, etc.It currently does not track downd activities to the smallest milisecond like trueSight, but rather just ticks of how many hapened

How to use preemtpive

When you open any of the app setup for preemptive (which is mostly all the apps) and run through it all the way to summary and print, preEmptive keeps track of that regardless of the environment. PreEmptive tracks all environment, you just have to make sure you are pointing preEmptive to the correct environment by using the correct link, as there is a preemptive link per environment
