Mockito

A java framework we use to perform Junit testing with.

Highlights

When we test functions that also calls other functions, we can use Mockito to return values to those fields.

  • Mockito.when(funcCallB(), ("value1", "value2", "value3") );
  • Each time the testing function is called, it will iterate through the 3 values you give it

Tips

  • IMPORTANT NOTE: When you Mockito.mock(Object.class) an object, you CANNOT set properties to that object or decorate it, although it may sound like you can, the mock object will never have properties
  • you cannot call multiple methods chains like this in mockito instead split chain method calls inside private methods into individual calls
    • ex: if you get this below inside a private method 
      Mockito.when(profile.getModel().getProducts( ProductType.BEAM )).thenReturn(beamProducts);
    • You want to break it into two when calls 
      Mockito.when(profile.getModel()).thenReturn(deckModel);
Mockito.when(deckModel.getProducts( ProductType.BEAM)).thenReturn(beamProducts);
