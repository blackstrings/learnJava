Change from sockets to web services 5/20/2015

So after going through several rings of fire (certs and firewall changes), Roofing and Siding estimator applications will work for Local, Dev and Stage. Hopefully prod too when the time comes. But there is one catch. With the new web services for creating estimates, stage and dev (local is probably set to stage for your PCs) can no longer use actual store numbers when using the override technique. If you try then you will get estimate ID of 0 because the new web service non prod will actively block it. So for dev and stage you will have to use 8888, 5555 or 4444.
