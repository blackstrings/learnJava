Mentools

A C++ file. Stands for Menards tools and is written in C++. So it is a helper file for our C++ applications. It serves as a way to connect to POS for pricing.

Applications Memtools is used in

  • Fence
  • Postframes
  • Garage

MemCache

MenCache uses Mentools for live pricing. What it does is cache live pricing into an internal data on one call to the Pricing server. Then the application can just from there on internally to get prices for less overhead.
