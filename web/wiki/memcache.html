Memcache

A local tool we run it to cache our C++ apps so that data loads faster when running the apps. Without memcache running in the background, you will see that your C apps load product information very slow.

Once loaded prices may change as new prices get updated in a way.

Highlights

  • Get and save the memcache.exe application to your desktop. Run it when running C++ apps.
  • Kill memcache if you need to test different stores and re-run it so you get a fresh cache. Do this by going to task manager > processors > memcache > kill it
  • You will need to restart memcache or run it each time you restart your computer
