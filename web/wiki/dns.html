DNS

Allows your utility applications to use Sys1 information of a store based on the store number. Ex: 3011=Eau Claire West. You have to set the DNS on your local computer. Internet needed.

Windows 7 > Control Panel > Network & Sharing > Local Area Connection > Properties > TCP/IPv4 or IPV6 > Properties > Advanced > DNS > Append these DNS suffixes (in order) > put in the addresses below

DNS Connections

  • menards.net
  • 3011.st.menards.net

Links with what tools

After setting DNS correctly you can use the PopConfig.exe which will generate the store.INI file.
