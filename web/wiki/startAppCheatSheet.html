Start App Cheat Sheet

If the app isn't running here is a check list.

  • Are you running java 7.45 - this is 32bit and our apps run 32 bit only
  • Don't use 7.67 as that is a 64bit and may not work
  • Check if the pom are pointing to the correct versions
  • Did someone change the environment urls and are you pointing to the correct URL ex: stage.ua-designit...
  • Is the environment in a 1-12-SNAPSHOT folder
  • Is the server down
