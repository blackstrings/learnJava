New Database (2015+)

We are moving to a new database as of 2015. The new database server is called intermiscdb. Our standsalone apps like Deck, landscaping will use this database. The backend wiring up database to the app is done through the designConfig envrionmet properties. Accessing the datbases directly, we can use heidi

Basics

Logins

Stage and prod share the same login. For Prod, each person has a unique loging, we use specific individual logins and these are not our active directory login. Prod is read only.

Connection Setup

Dev and Stage share the same logins, just different ip host. Dev is intermiscdb-d.menards.net, and stage is intermiscdb-s.menards.net Using heidi, use TCP, not ssh tunnel.
