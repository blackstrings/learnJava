Flooring

A stand alone app. This application allows users to create floorings for their rooms. You can only choose up to 4 rooms in a given app session.

Unique features the app supports

  • Prints out multiple papers sometimes too much (un-sufficient in the way)
  • A printout of how to cut the floorings to fit the specified room dimensions
  • Create openings in a room and have floorings estimate for that opening.
  • Pay right at the kiosk at the end

Setup

  • Eclipse
  • Java
  • Dependencies: REAF, 3D, Common, Deck

MISC

  • JIMGUI (search and find created orders)
