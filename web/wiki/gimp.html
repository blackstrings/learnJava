Gimp

A free photo editor tool similar to photoshop, just not as robust. However, it does have some powerful tools once you learn the non-intuitive interface.

Shortcuts

Paths
Ctrl + click With a path pre-selected, click on the start path to close the path
Alt + click With a path pre-selected, click on any of the path points or segments to move the path as a whole
Shft + click Select multiple path points

Hints
Creating a stroke on path Create a closed path, click on brush itcon under path panel to add a stroke around path. Use paint bucket after stroke to paint in.

Cropping Image Layer

Gimp is not like photoshop where you can auto crop an image to its layer boundary. Instead, use the square selections tool > Layer > crop to selection
