Updating Java Screens

Screens in Java apps are generated through XML files. To create new or edit current screens, manipulate XML files under src/main/resources/xml/selections/xxxScreen.xml of each project workspace.

Important Tags

  • <AutoPanel> - display objects on screen that will be dyanmic, i.e. selections
  • <FixedPanel> - display objects on screen that are fixed
  • <Button> - used to hold a place for one object or action
  • <Image> - to show image
  • <Text> - text box to show text on screen
  • Actions or Parameters
    • <Location> - location on the screen in relation to the object it is in.
    • <Size> - boundary of the object, can be full screen sometimes
    • <IsEnabled> - condition if a button should be shown as enabled or disabled
    • <Enabled> parameters or objects inside this tag are displayed if IsEnabled returns as true
    • <Disabled> - parameters or objects inside this tag are displayed if IsEnabled returns as false
    • <onDown> - anything inside this tag is performed when the Object is clicked or selected
    • <script> - run lua script after opening tag and before the end tag. In other words run Lua script at place of tag insert.
