Notepad++

An advance text editor usful for programming source codes. Its unique text handling makes it a valuable tool for programmers to open and view source codes in whether the language is Java, C++, etc.

Highlights

  • Very handy for editing java XML files and C++ INI files
  • Many languages to choose from under language settings
  • Good for copy/paste code into microsoft word doc for Business Design

Copy code onto word documents and keep formating

  • copy code from IDE and past into new notepad++ doc
  • Language > select desired programming language
  • Highlight/Select the text you want copied over
  • Notepad++ > Plugins > NppExport > Copy all formats to clipboard > paste over to Microsoft word doc
