Store Support

If you need to investigate an issue with the kiosks at a particular store there are a few ways to go about it. First check what store # it is through using myMenards lookup. Then use filezilla to get the PC's dns or ip. You can use pcAnywhere (start > all programs > symantec > pcA Quick Connect) Instructions into it and check if the pc has the correct version installed.

Past issue example: the kiosk was not displaying the contents correctly due to the PC not haveing enterprise mode installed/enabled which seems to be an imaging thing. So imaging was emailed to install enterprise mode on these PCs.

Knowing what pc is what OS through C$ or FileZilla

C:\windows\system32\ > right click on ntoskrnl.exe > properties > details > product version >

  • 6.3 = Windows 8.1
  • 6.2 = Windows 8
  • 6.1 = Windows 7
  • 6.0 = Windows Vista
  • 5.2 = Windows Server 2003
  • 5.1 = Windows XP
  • 5.0 = Windows 2000
  • 4.0 = Windows NT 4
