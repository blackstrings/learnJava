Jenkins

An online site that we use to build our applications for testing purposes. Jenkins will run through our junits and check for findbugs.

Highlights

You have to log in to be able to browse through applications

The server will continuously run our test codes and inform us of any errors that may occur when we do code updates. Jenkins is related to Junit in a way. If we know how, we are suppose to run our code changes/app against jenkins after code review

When does jenkins build our app?

If you already have a MCR created in jenkins, and you update the MCR by pushing the MCR to stash or to the repo, jekins will pick this up and auto start building the app and running it throgh findbugs. You can also force build if there isn't a build currently being built. If you dont' already have an MCR register in jenkins, follow the steps below.

Findbugs

During a build in jenkins, jenkins will use its findbug tool to catch for errors in your application and alert you through its online interface.
